new classes TFitResult, TFitResultPtr


    
 
  •  All the Fit method of the Hist library now instead of returning an integer returns a TFitResultPtr object. The TFitResultPtr is an object that converts automatically to an integer, the status code of the fit. There is a no visible change for the user, if the Fit method is used as before.
  
  • When using the fit option "S", the TFitResultPtr will contain a pointer to the new TFitResult class. In addition it will behave as a smart pointer to TFitResult, so by using the -> operator the user can call the TFitResult methods or by using the de-reference operator * or Get(), access directly the TFitResult object. 
  
  • TFitResult derives from the ROOT::Math::FitResult which contains all the result information from a fit and from TNamed. It provides then I/O capabilities for the FitResult object and conevenience methods like Print(), Write() and GetCovarianceMatrix() and GetCorrelationMatrix() which return a TMatrixDSym object.
  
  • Example of usage:
  
         TFitResult r = h->Fit("myFunc","S");
     TMatrixDSym cov = r->GetCovarianceMatrix();  //  to access the covariance matrix
     Double_t chi2   = r->Chi2(); // to retrieve the fit chi2 
     Double_t par0   = r->Value(0); // retrieve the value for the parameter 0 
     Double_t err0   = r->Error(0); // retrieve the error for the parameter 0 
     r->Print("V");     // print full information of fit including covariance matrix
     r->Write();        // store the result in a file
